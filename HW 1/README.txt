Explanation of files:

1. "imagefeatures" folder is the raw dataset
2. "spring_summer_autumn" folder contains only spring, summer, and autumn images (1145 images which are the subset of raw dataset)
3. Inside "spring_summer_autumn" folder are 3 folders titled "train", "validate", and "test". Inside each of those folders are the
   .npy files allocated for training, validation, and testing respectively.
4. All python codes & analysis for the two accuracies (vanilla accuracy & class wise averaged accuracy) are available inside
   the .ipynb jupyter notebook file.
